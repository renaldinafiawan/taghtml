Elemen Semantik HTML5

HTML5 memperkenalkan elemen semantik yang memberikan makna pada struktur halaman web:

  • <header> - Bagian header halaman atau bagian
  • <nav> - Bagian navigasi
  • <main> - Konten utama halaman
  • <section> - Bagian dari dokumen
  • <article> - Konten independen yang dapat berdiri sendiri
  • <aside> - Konten sampingan yang terkait dengan konten utama
  • <footer> - Bagian footer halaman atau bagian
  • <figure> dan <figcaption> - Untuk gambar dengan keterangan
<header> <h1>Judul Website</h1> <nav> <ul> <li><a href="#">Beranda</a></li> <li><a href="#">Tentang</a></li> <li><a href="#">Kontak</a></li> </ul> </nav> </header> <main> <section> <h2>Bagian 1</h2> <article> <h3>Artikel 1</h3> <p>Konten artikel...</p> </article> </section> <aside> <h3>Sidebar</h3> <p>Konten sampingan...</p> </aside> </main> <footer> <p>© 2023 Website Saya</p> </footer>

Atribut HTML Penting

Beberapa atribut HTML penting yang sering digunakan:

  • id - Mengidentifikasi elemen secara unik
  • class - Mengklasifikasikan elemen untuk styling CSS
  • style - Menambahkan CSS inline
  • title - Menambahkan tooltip saat hover
  • data-* - Menyimpan data kustom
  • aria-* - Atribut aksesibilitas

Contoh penggunaan:

Elemen dengan berbagai atribut
<div id="contoh-id" class="contoh-class" style="color: blue;" title="Ini adalah tooltip" data-info="Informasi tambahan" > Elemen dengan berbagai atribut </div>

Karakter Khusus (Entities)

Beberapa karakter khusus dalam HTML:

  • &lt; menghasilkan: < (tanda kurang dari)
  • &gt; menghasilkan: > (tanda lebih dari)
  • &amp; menghasilkan: & (ampersand)
  • &quot; menghasilkan: " (tanda kutip ganda)
  • &apos; menghasilkan: ' (tanda kutip tunggal)
  • &copy; menghasilkan: © (copyright)
  • &reg; menghasilkan: ® (registered trademark)
  • &trade; menghasilkan: ™ (trademark)
  • &nbsp; menghasilkan: spasi yang tidak terputus
